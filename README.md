# E Royal Mango — Frontend

Next.js 15 storefront for E Royal Mango.

## Development

```bash
npm run dev
npm run dev:clean   # clear cache + dev
```

## Structure

| Directory | Purpose |
|-----------|---------|
| `app/` | Routes and layouts |
| `components/` | UI components |
| `sections/` | Marketing homepage sections |
| `services/` | Data modules (static until API) |
| `store/` | Cart and wishlist state |
| `styles/` | Global CSS |

Path alias: `@/*` → package root.
